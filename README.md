# sph(Vue3)

## Description
This project is the [Vue 3](https://vuejs.org/) edition of [sph](https://github.com/gengzhao2020/sph). Besides, it comes with [Vite](https://vitejs.dev/) and [Pinia](https://pinia.vuejs.org/).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Locally preview production build
```
npm run preview
```
